# 最优化实验室 (OptLab) 项目概述

## 项目简介
《最优化实验室》是一个专注于“最优化理论与方法”的教学实验平台。项目旨在通过交互式的可视化实验，帮助学习者直观理解各种优化算法的原理、流程及特性。

## 技术栈
- **前端**: HTML5, CSS3, JavaScript
- **可视化库**: 
  - **D3.js**: 主要用于二维函数图像绘制、迭代过程动画展示（如一维搜索、单纯形法表格动态更新）。
  - **three.js**: 计划用于三维函数曲面及高维优化问题的可视化演示。
- **后端**: **FastAPI** (Python 3.13+)，目前处于起步阶段，仅包含基础框架。
- **包管理**: **uv** (使用 `pyproject.toml` 和 `uv.lock`)

## 目录结构说明
- **resources/**: 核心网页存储目录，按算法类别组织。
  - `index.html`: 平台主页（待完善）。
  - `1d_search/`: 一维线搜索实验，包含区间搜索（黄金分割法、斐波那契法）等。
  - `gd/`: 梯度下降法相关实验。
  - `lp/`: 线性规划相关实验。
- **demo/**: 实验性与功能演示文件。
  - `1d_demo.html`: D3.js 效果测试。
  - `simplex_demo.html`: 单纯形法交互式演示（含动态表格与步骤追踪）。
- **docs/**: 项目文档。
  - `overview.md`: 本文档（项目全局概述）。
  - `最优化内容概览.pdf`: 理论参考资料。
- **temp/**: 临时文件与素材。
- **main.py**: FastAPI 后端服务入口。
- **pyproject.toml / uv.lock**: 依赖配置与锁定文件。

## 已实现功能
- **一维线搜索可视化**: 实现了黄金分割法和斐波那契法的区间缩减过程演示，支持自定义函数输入和单步迭代。
- **单纯形法交互演示**: 提供了单纯形表的手动/自动迭代演示，突出显示转轴行、转轴列及入基/出基过程。

## 开发计划 (2月目标)
1. **完善前端交互**: 统一 `resources` 下各模块的 UI 风格，增强 D3.js/three.js 的动画表现力。
2. **后端服务开发**: 基于 FastAPI 实现算法计算接口，支持复杂优化问题的后端求解并返回前端展示。
3. **集成与部署**: 实现静态资源的自动化路由，搭建完整的本地/在线实验环境。

## 开发者指南
- 前端开发建议参考 `resources/1d_search/range_search` 目录下的脚本组织方式（scripts 分离）。
- 后端扩展请在 `main.py` 中定义 API 路由，并遵循 FastAPI 异步处理规范。
- 依赖安装: `uv sync`
